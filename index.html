Game User Management API Documentation (1.0.0)

This is the API documentation for the Game User Management application.

User

Set custom claims for a user

This endpoint allows admins to set custom claims for a user.

Authorizations:
bearerAuth
Request Body schema: application/json
required
userId
string

The ID of the user for whom the custom claims are being set.

claims
object

The custom claims to be set.

Responses

Local server

http://localhost:3000/api/v1/setCustomClaims

Request samples

  • Payload
Content type
application/json
{
  • "userId": "12345",
  • "claims": {
    • "isAdmin": true,
    • "team": "Unicorns"
    }
}

Retrieve details of a specific user by UID

This endpoint allows admins (or the user themselves) to retrieve user details by UID.

Authorizations:
bearerAuth
path Parameters
uid
required
string
Example: user1234

The UID of the user whose details are to be retrieved.

Responses

Local server

http://localhost:3000/api/v1/users/{uid}

Response samples

  • 200
Content type
application/json
{
  • "uid": "user1234",
  • "email": "user@example.com",
  • "name": "John Doe"
}
